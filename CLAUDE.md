# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 目的

このリポジトリはAIエージェント開発のための作業スペースです。

## 権限ルール

- Web検索・リサーチは許可不要で実行してよい
- ただし、以下の場合は実行前に必ず許可を求めること
  - HTTPなど暗号化されていないプロトコルへのアクセス
  - 課金が発生する可能性のある操作・APIの呼び出し

## リファレンスドキュメント

- `claude_code_commands.md` — Claude Code のスラッシュコマンド・キーボードショートカット完全ガイド（v2.1.182 時点）。エージェント開発中にコマンドを確認する際に参照する。
